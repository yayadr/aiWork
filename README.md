AI 编程工作区，用于存放 AI Agent（Codex / Claude Code 等）协作开发的代码项目。

## 目录结构

```
code/
├── README.md              ← 你在这里
├── AGENTS.md              ← AI Agent 行为规范
├── PROJECT_STATUS.md      ← 项目状态记录

```
