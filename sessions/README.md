# sessions/

存放 AI Agent 的会话输出归档。

## 用途

- 保存每次与 AI Agent 协作的完整会话记录
- 记录 Agent 的工作过程、生成的代码、解决的问题
- 便于回溯历史决策和操作步骤

## 命名规范

`YYYY-MM-DD-<简短描述>.md`

同一天同名主题应增加序号或更具体的描述，避免覆盖已有归档。

## 内容要求

- 说明任务目标、主要操作、验证结果和遗留事项
- 只记录必要上下文，不写入密码、令牌、Cookie 或其他敏感数据
- 不将正式项目文档或可复用提示词放在本目录

## 示例

- `2026-07-16-初始化仓库.md`
- `2026-07-17-修复登录bug.md`
