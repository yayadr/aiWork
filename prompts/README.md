# prompts/

存放可复用的 AI 提示词模板。

## 用途

- 沉淀经过验证的高效提示词
- 按任务类型分类管理
- 团队成员共享优秀提示词

## 结构

```
prompts/
├── coding/          # 代码生成类提示词
├── review/          # 代码审查类提示词
├── debug/           # 调试排错类提示词
├── doc/             # 文档生成类提示词
└── design/          # 架构设计类提示词
```

## 模板格式

```markdown
# <提示词名称>

**适用场景**：<描述>
**模型推荐**：<模型名>
**标签**：<tag1, tag2>

---

<提示词正文>
```
